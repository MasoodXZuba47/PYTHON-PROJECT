# ATM Simulation (Python)

A simple **console-based ATM simulation** written in Python.  
This beginner project demonstrates basic Python concepts such as variables, conditional statements, and user input.

---

## Features
- Check account balance
- Deposit money
- Withdraw money with balance validation
- Simple menu-driven interface

---

## Tech Stack
- Python 3
- No external libraries

---

## How the Program Works
1. The program starts with a predefined balance
2. A menu is displayed:
   - Check Balance
   - Deposit Money
   - Withdraw Money
3. User selects an option by entering a number
4. The program performs the selected operation
5. The program ends after one operation

---

## Code Overview
- Initial balance is stored in a variable
- `if / elif / else` is used for menu handling
- User input is taken using `input()`
- Balance updates dynamically based on user actions
